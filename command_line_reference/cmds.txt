======================
Starting Fluent
======================

- Open 3d fluent with double precision input with no gui:
> fluent 3ddp -g

- run fluent case file (TUI commands)
> fluent 3ddp -g < input-file-name > & input-file-name &

- exit fluent
> exit
> yes

======================
IO
======================

- load case file
> file/read-case <file-name>

- write case file
> file/write-case <file-name>

- write data file
>file/write-dat <file-name>

- add report file
	> solve/report-files/add <file-name>
	- add definitions to report file
		> <report-def-name>
		> <report-def-name>
	
- get properties/report-definitions
	> solve/report-definitions/list
	
======================
Solver
======================

- initialization
	- hybrid-initialization
		> solve/initialize/hyb-initialization
	- regular-initialization
		> solve/initialize/initialize-flow
		
- set number of iterations
	> solve/set/number-of-iterations [float] - default 150
	
- iterate solution
	> solve/iterate [float] - default 500
	
- convergence criteria:
	> solve/monitors/residual/convergence-criteria
		> continuity residual convergence criterion [float] - default [0.0001]
		> x-velocity residual convergence criterion [float] - default [0.0001]
		> y-velocity residual convergence criterion [float] - default [0.0001]
		> z-velocity residual convergence criterion [float] - default [0.0001]
		> energy residual convergence criterion [float] - default [1e-07]
		> k residual convergence criterion [float] - default [0.0001]
		> epsilon residual convergence criterion [float] - default [0.0001]
